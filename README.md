# To-Do List

## What it Does

This is a simple Python console application that helps users manage a to-do list. Users can add tasks, remove tasks, and view all saved tasks through a menu-driven interface.

## How to Run It

1. Make sure Python 3 is installed.
2. Clone this repository:
   ```bash
   git clone https://github.com/Belalmohamed448/To-Do-List.git
   ```
3. Open the project folder.
4. Run the program:
   ```bash
   python todo_list.py
   ```
5. Use the menu to add, remove, or view tasks.
